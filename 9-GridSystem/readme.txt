
Responsive design: sviluppare pagine web che ben siano mostrate su dispositivi aventi larghezze di schermo differenti
Il responsive design prevede 2 approcci:
- mobile first => media query con min-width
- desktop first => media query con max-width


Nel responsive design con approccio mobile first:
1) sviluppare la pagina prima (in particolare il layout) in modo che sia ben mostrata su mobile orientanto in portrait (cioè in verticale)
2) il numero "MINIMOOOO" di modifiche al CSS affinchè la pagina sia ben mostrata per larghezze di schermo via via crescenti passando da:
- mobile in landscape (cioè in orizzontale)
- tablet in portrait (cioè in verticale)
- tablet in landscape (cioè in orizzontale)
- desktop
- smart tv
