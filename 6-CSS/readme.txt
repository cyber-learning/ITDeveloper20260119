Pseudo-classi
    :hover
    :focus
    :root

La Pseudo Classe :root è utile per 2 motivi:
- dichiarare le variabili (globali) del CSS
- dichiarare un font-size di base

Unità di misura per i font:
- assolute px
- relative em rem (root em)
